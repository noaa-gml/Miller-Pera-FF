#   python -m pip install -r requirements.txt
#
# Frozen to the exact versions the v2026b product was built and validated
# against (May 2026). Exact pins are deliberate: an unpinned dependency
# that shifts a binary ABI is how a silently-wrong numerical output
# happens — this is the lesson of the PyFerret numpy-ABI breakage. The
# versions that produced an output file are also recorded inside the file
# itself (see provenance.py).
#
# SCOPE: this is the pip-installable subset. It is what CI installs, and is
# enough for the lint / type-check / test gates and every pipeline stage
# EXCEPT the EDGAR 0.1deg -> 1deg regrid. Three dependencies are conda-forge
# only and are NOT listed below; for a complete, runnable environment use
# conda, which installs them at these same pinned versions:
#
#   conda env create -f environment.yml      # canonical, fully reproducible
#
#     xcdat  0.11.2   (not published to PyPI)
#     xesmf  0.8.8    (binds the compiled ESMF library)
#     esmpy  8.9.1    (compiled ESMF library)
